# DTL
Data Template Library

## 容器
* array：静态数组
* vector：动态数组
* forward_list：单向循环链表
* list：双向循环链表
* deque：双端队列
* avl_tree：平衡二叉树

## 容器适配器
* heap：堆 -> vector
* queue：队列 -> deque
* priority_queue：优先队列 -> heap
* stack：栈 -> vector
* set：单重有序集合 -> avl_tree
* multiset：多重有序集合 -> avl_tree
* map：单重有序映射 -> avl_tree
* multimap：多重有序映射 -> avl_tree
